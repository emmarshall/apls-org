About The Project

An NHS Theme for deploying Quarto websites to GitHub.io pages with automated GitHub action.

Note: No data, public or private are shared in this repository.

Folder Stucture

Name Link Description
.github/workflows [Link] Github Action workflow files that automate the publishing process
docs/ [Link] All files for rendering the Quarto website
docs/_assets/ [Link] Art and style assets that build the page theme
docs/documentation [Link] All markdown files here will be added to the documentation sidebar
docs/posts [Link] Blog posts. Each post should be in its own folder as a index.qmd file
talks [Link] All markdown files here will be added to the tutorial sidebar
